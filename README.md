# SPAM
SPAM (Speed Up PAM) 是一种加速 K-medoids 聚类算法，由 余冬华、郭茂祖 等人于 2017 年提出。

### Paper Information
余冬华，郭茂祖，刘扬* ，任世军，刘晓燕，刘国军. **基于距离不等式的 K-medoids 聚类算法**. _软件学报_. 2017,28(12):3115-3128.

### Code and Data
:
- 编译 'SpeedUpPAMCount.h', 'SpeedUpPAMCountmain.cpp'，然后运行
- 需要在运行时输入数据文件名称
